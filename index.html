HTML & CSS CHEAT SHEET

HTML

HTML Table Tags

Tag Name Use
<table> Table Initiate a table
<thead> Table Header The header of the table and the rows within it
<tbody> Table Body The body of the table and the columns of rows within it
<tr> Table Row Wrapper for each row in a table
<th> Table Header A column header in a table
<td> Table Data what goes in a specific table cell
<tfoot> Table Footer Rows that are in a table footer

More HTML Tags

Tag Name Use
<h1> Section Headings Creates a heading. h1-h6, size in descending order.
<p> Paragraph Write text in paragraph form.
<img> Image Add an image

CSS Properties

Style Use
color change color of text

text-align where you want the text (center, etc.)
